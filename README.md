## Messenger

### Описание

---

- Проектная работа для курса Яндекс Практикум
- Приложение для обмена сообщениями


### Установка

---

Сборка и запуск проекта:
```bash
npm run start
```

Запуск проекта в режиме разработки:
```bash
npm run dev
```
